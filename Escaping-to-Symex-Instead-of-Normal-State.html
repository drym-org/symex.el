6.1 Escaping to Symex Instead of Normal State ¶

For Evil users, when you “escape” from Insert state, you may prefer to enter Symex state rather than Normal state while in Lisp buffers.

Please See Symex and Rigpa for the recommended way of achieving this.

If you’re not using Rigpa, you could still write one-off keybindings to achieve this, as described in See Easy Entry Into Symex Mode.
