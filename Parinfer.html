1.3.8 Parinfer ¶

Parinfer exploits a duality between indentation and parenthesis nesting to handle either for you when you change the other.

You should be able to use this together with Symex. If you do, you will likely want to set symex-tidy-after-transforming-p to nil, and possibly also symex-ensure-structure-p to nil. See Customizing Symex for more information.
