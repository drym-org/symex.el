2.6.1 Which Packages Should I Install? ¶

If you aren’t an Evil user, using symex-core, symex, and symex-ide together provide the same (but improved) experience as before. If you are an Evil user, you would additionally install symex-evil to get the same experience as before. If you are a Rigpa user, install symex-rigpa in addition, for the same experience as before.
